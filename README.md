# 🔥 Calorie Master — AI Food Tracker

An AI-powered calorie tracking and nutrition analysis web app built with React, Firebase, and Google Gemini AI.

## ✨ Features

- **AI Food Scanner** — Snap a photo of your meal and get instant calorie & macro breakdowns
- **BMR Calculator** — Calculate your exact daily caloric needs with multiple formulas
- **Daily Tracking** — Track progress with animated charts and streak system
- **Recipe Suggestions** — AI-generated healthy recipes from your ingredients
- **Water Tracking** — Log daily water intake
- **Multi-language** — Full English and Arabic (RTL) support
- **PWA** — Installable as a mobile app

## 🛠 Tech Stack

- **Frontend**: React 19, TypeScript, TailwindCSS v4, Framer Motion
- **Backend**: Firebase Auth, Firestore
- **AI**: Google Gemini 3 Flash
- **Build**: Vite, PWA support

## 🚀 Getting Started

```bash
npm install
npm run dev
```

## 📝 Configuration

1. Create a Firebase project and add your config to `src/firebase.ts`
2. Set up Firestore security rules from `firestore.rules`
3. Add your Gemini API key in the Profile page after logging in
